Our Goal

This game has the goal to teach everyone who plays it main musical notes of different percussion instruments. As well to raise awarness
on the health issues that playing a percussion instrument can have. We provide both a fun and interactive game to play while learning to
play notes on different musical instruments. This game is intended to raise awarness on these subjects.

Gameplay

Drummy is a percussionst player who has to use his drummer's skills to save his fellow comrades. While doing so Drummy can teach them tips to prevent injuries while playing these types of instruments.

Learning Process

This game teaches the user some of the main perscussion instruments. It also raises awarness on the injuries that can occur from playing any percussion instruments and the main means to avoid them.

Levels

The game has three levels, each one more difficult. Drummy has to play the right notes to be able to solve each puzzle. The game shows the main notes of three precussion instruments.

Score

Be the best by saving all your friends the fastest way you can. Drummy has to go through all the levels to finish the game and save all his comrades. The scores of all the players are shown in top scores. Can you be the best?
